esgf-map
--------

Procedure to generate ESGF KML file
***********************************

.. code-block: bash
    $> pip install pyessv
    $> mkdir ~/.esdoc/
    $> git clone https://github.com/glevava/pyessv-archive.git ~/.esdoc/.
    $> git clone https://github.com/ESGF/esgf-map.git
    $> cd esgf-map
    $> python esgf_map.py
